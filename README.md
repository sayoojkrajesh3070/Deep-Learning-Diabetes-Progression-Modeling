# Deep Learning: Diabetes Progression Modeling

This repository contains the completed module-end assignment for the Deep Learning course. The project involves building and improving an Artificial Neural Network (ANN) to model the progression of diabetes using the `sklearn` Diabetes dataset.

## Assignment Overview
The goal of this project is to model diabetes progression using independent clinical variables. This analysis provides insights into the dynamics of diabetes progression, which can assist in designing treatment plans and preventive measures.

## Key Components Fulfilled
The project is structured into the following sections as per the assignment requirements:
* **Loading and Preprocessing**: Data retrieval, checking for missing values, and feature normalization.
* **Exploratory Data Analysis (EDA)**: Visualizing feature distributions and relationships with the target variable.
* **Building the ANN Model**: Designing a simple ANN architecture with at least one hidden layer and appropriate activation functions.
* **Training the ANN Model**: Implementing a train-test split, utilizing suitable loss functions, and optimizing model weights.
* **Evaluating the Model**: Assessing performance using Mean Squared Error (MSE) and $R^2$ Score.
* **Improving the Model**: Experimenting with hyperparameters and architecture changes to enhance performance.

## Technologies Used
* **Language**: Python
* **Environment**: Google Colab / Jupyter Notebook
* **Libraries**: TensorFlow/Keras, Scikit-learn, Pandas, NumPy, Matplotlib, Seaborn
